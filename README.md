# Disclaimer
Este proyecto tiene fines estrictamente educativos y de prototipado. Al construir o utilizar el Merlin Bot, usted acepta lo siguiente:

Riesgo Eléctrico: El uso de baterías de Li-ion (18650) requiere precaución. Un cortocircuito o polaridad invertida puede causar daños permanentes a los componentes o riesgos de incendio.

Voltaje Crítico: Es responsabilidad del usuario ajustar el módulo LM2596 a 5.0V mediante un multímetro ANTES de conectar el ESP32. El autor no se hace responsable por microcontroladores dañados por sobrevoltaje.

Uso "Tal Cual": El software y los diagramas se proporcionan "as is" (tal cual), sin garantías de ningún tipo. El autor no se hace responsable de daños materiales o personales derivados del ensamblaje o uso del robot.

Supervisión: Se recomienda que menores de edad realicen el ensamble bajo la supervisión de un adulto o instructor capacitado.
