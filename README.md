### ⚠️ ADVERTENCIA
> El proyecto **Merlin Bot** involucra el uso de componentes electrónicos y baterías de litio. Al construir o utilizar este robot, usted acepta lo siguiente:
> * **Voltaje Crítico:** Es **obligatorio** ajustar el módulo **LM2596 a 5.0V** mediante un multímetro **ANTES** de conectar el ESP32. El autor no se hace responsable por microcontroladores dañados por sobrevoltaje.
> * **Riesgo Eléctrico:** El uso de baterías Li-ion (18650) requiere precaución. Una polaridad invertida o cortocircuito puede dañar los componentes o representar un riesgo de incendio.
> * **Uso "Tal Cual":** Este proyecto se proporciona con fines educativos y "as is" (tal cual). El autor no asume responsabilidad por daños materiales, personales o pérdida de componentes derivados del ensamblaje o uso del robot.
> * **Supervisión:** Se recomienda que menores de edad realicen el ensamble bajo la supervisión de un adulto o instructor capacitado.
