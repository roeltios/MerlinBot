# Merlin Bot Nivel 2: Percepción y Reacción (Sensores)

En el segundo nivel, Merlin Bot adquiere la capacidad de percibir su entorno. Se introduce el concepto de retroalimentación (feedback) para la toma de decisiones autónomas.

## Adiciones de Hardware
* **Sentido de Proximidad:** Sensor Ultrasónico HC-SR04.
* **Soporte de Montaje:** Base frontal para el sensor.

## Objetivos Pedagógicos
1. **Entrada de Datos:** Lectura y procesamiento de señales externas (distancia en cm).
2. **Lógica Condicional:** Implementar algoritmos de "Si ocurre X, entonces haz Y" (Evasión de obstáculos).
3. **Depuración (Debugging):** Usar la pantalla OLED para visualizar en tiempo real lo que el robot "ve" antes de actuar.

## Segundo Reto
Crear un algoritmo de navegación autónoma donde Merlin recorra una habitación sin colisionar, usando la pantalla para alertar cuando detecte un objeto cercano.
